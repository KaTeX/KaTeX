## npm
Install with `npm`:

```bash
npm install katex
# or globally
npm install -g katex
```

## Yarn
Install with `Yarn`:

```bash
yarn add katex
# or globally
yarn global add katex
```
<br>
> You can manually download the package from [here](https://github.com/Khan/KaTeX/releases)
